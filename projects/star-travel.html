Star Travel

Animação com câmera viajando pelo sistema solar, construída em OpenGL e C++.

OpenGL C++ 3D Status: preciso melhorar muito o código ainda.

Gif

Resumo do projeto

Objetivo

Praticar conceitos matematicos e voltados para o OpenGL como os cálculos envolvidos na movimentação, desenvolvimento da perspectiva e geração dos modelos 3D.

Desafios

A curva de aprendizado fora realmente bem difícil. São muitas funções prontas, uma biblioteca muito extensa e complexa, bem como a forma de passar os conceitos teóricos para o código também foi um grande desafio.
